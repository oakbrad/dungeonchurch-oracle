How to Embed

To embed the Dungeon Church Oracle visualization in your own website, use the following HTML code:

<div style="width: 100%; height: 500px; overflow: hidden;">
  <iframe src="https://oakbrad.github.io/dungeonchurch-oracle/" width="100%" height="100%" frameborder="0"></iframe>
</div>

You can adjust the width and height of the container to fit your layout. The visualization will automatically adapt to the available space.

Advanced: Communicating with the Embedded Visualization

If you need to notify the visualization when its container has been resized (for example, in dynamic layouts), you can use the postMessage API:

// Get a reference to the iframe
const iframe = document.querySelector('iframe');

// Send a resize message to the visualization
iframe.contentWindow.postMessage({ type: 'resize' }, '*');

This can be useful when the container size changes due to user interactions or dynamic content loading.
